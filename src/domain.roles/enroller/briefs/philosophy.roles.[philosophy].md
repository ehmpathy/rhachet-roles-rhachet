# philosophy: roles

## .what

roles crystallize knowledge into reusable, shareable forms.

```
🧢 role = 📚 briefs + 💪 skills
```

- **briefs** = crystallized concepts (rules, guides, lessons, patterns)
- **skills** = crystallized capacities (executable operations)

## .why

brains learn, but the cost is high. every insight discovered must be:
- recognized as valuable
- articulated clearly
- tested against edge cases
- refined through use

this cost is paid once per discovery. without crystallization, the same cost is paid by every brain, every session, every context.

roles amortize learned knowledge across brains.

## .the value proposition

| without roles | with roles |
|---------------|------------|
| each brain discovers patterns | patterns are pre-crystallized |
| knowledge lost between sessions | knowledge persists in briefs |
| capacities rebuilt each time | capacities encoded in skills |
| quality varies by luck | quality is institutionalized |
| improvement is local | improvement is global |

## .crystallization

### briefs crystallize concepts

a brief captures a concept that was:
- discovered through experience
- validated through use
- refined through iteration

once crystallized, the concept is available to any brain that enrolls the role.

```
discovery → articulation → validation → crystallization → reuse
```

### skills crystallize capacities

a skill captures a capacity that was:
- implemented correctly
- tested thoroughly
- documented clearly

once crystallized, the capacity is available to any brain that enrolls the role.

```
implementation → test → documentation → crystallization → invocation
```

## .iteration

crystallization is not static. roles improve over time:

1. **use reveals gaps** — brains encounter edge cases briefs don't cover
2. **gaps become briefs** — new insights are crystallized
3. **briefs inform skills** — patterns become executable
4. **skills enable new use** — capacities unlock new applications

each cycle compounds. the role becomes more capable, more refined, more valuable.

## .clone best practices

when a brain enrolls a role, it inherits:
- every brief the role has crystallized
- every skill the role has encoded
- every refinement prior brains contributed

this is knowledge clone. the brain starts with accumulated wisdom rather than from scratch.

## .the mandate

> crystallize concepts into briefs.
> crystallize capacities into skills.
> iterate. improve. compound.

roles are how we stop re-learn and start compound.
